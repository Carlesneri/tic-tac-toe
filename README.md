# tic-tac-toe
Play Tic Tac Toe agains computer

## Requirements
- npm ~7 (workspaces)
- node ~13 (for testing with ES6)

## Instructions
- Install dependencies: `npm i`
- Create dist folder: `npm run build`
- Start the server: `npm start`
- Visit: `http://localhost: 3001`

## Scripts
- build: `npm run build`
- start: `npm start`
